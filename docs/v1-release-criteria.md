# Criterios para 1.0

- Cero bugs bloqueantes o pérdida de datos conocidos.
- Exportación estable y equivalente al preview dentro de límites documentados.
- Round-trip completo de proyectos/propuestas en navegadores soportados.
- Estrategia principal de detección definida y validada con dataset real, o alcance manual comunicado.
- Privacidad/términos revisados para servicios realmente activos.
- Todos los assets publicados con licencia/procedencia confirmada.
- Móvil usable para el alcance declarado y gestos críticos en hardware.
- Chrome, Safari, Firefox y Edge verificados en Preview.
- Preview y Production validados con dominio, metadata, logs y rollback.
- Documentación, changelog, release notes, seguridad y canal de soporte completos.
- Rate limit/políticas Supabase adecuados al tráfico real si esos servicios están habilitados.

No se fija una fecha. 1.0 solo se publica cuando la evidencia cumple estos criterios.
